# Smart Task Manager API

## Tech Stack
- Java 17
- Spring Boot
- PostgreSQL
- Maven

## Features
- CRUD operations
- Filtering (priority, completed)
- Validation
- Global exception handling

## Run Locally
1. Configure database in application.properties
2. Run the application
3. Access API at http://localhost:8080/tasks

Frontend: https://github.com/christianmutiwadirwa-cryptochristianmutiwadirwa-crypto/task-manager-frontend
